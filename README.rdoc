# Projeto BojoFactory
## O projeto contempla uma fábrica de bojo no qual deve calcular quais pedidos devem ir
para produção de acordo com a quantidade total por pedidos em relação ao estoque

# Tecnologias
* Ruby on Rails
* PostgreSQL

# Como rodar o Projeto ?
## Após instalar Ruby rode o comando `bundle install` para instalar as dependências,
posteriormente rode o comando `rake db:create; rake db:migrate` para criar o banco dedados
e fazer as migrações.
